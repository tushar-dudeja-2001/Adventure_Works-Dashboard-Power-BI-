# Adventure_Works-Dashboard-Power-BI
This project contains an interactive Power BI dashboard built using the AdventureWorks dataset. It showcases business insights across sales, customers, products, and regional performance.
The report highlights key KPIs and supports data-driven decision-making.

Key Insights
  Total sales performance across time
  Top-selling products and categories
  Customer segmentation and demographics
  Profitability analysis
  Regional performance insights
  Trend analysis with DAX measures

Tools & Technologies
  Power BI Desktop
  DAX (Data Analysis Expressions)
  AdventureWorks Dataset
  Data Modeling & ETL inside Power BI
